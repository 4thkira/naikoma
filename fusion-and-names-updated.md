# Fusion and Names — Building Complex Meaning

Naikoma's power lives in what happens when roots stop being separate words and start becoming each other. Simple compounding (nai-ko = love-home) is the surface. Beneath it are fusion techniques that compress entire philosophies into single words — and a naming system where identity is a sentence you carry.

This document covers five layers of complexity, from basic compounding up through name construction. Each layer builds on the last.

## Layer 1: Compounding (Review)

Two roots placed side by side. The last root is the head (what the compound IS), earlier roots modify it (what KIND).

- **nai-ko** = love-home → "a home made of love"
- **ko-nai** = home-love → "the love one has for home" / homesickness
- **lume-kore** = fire-heart → "a heart made of fire"
- **kore-lume** = heart-fire → "a fire that comes from the heart"

Written with hyphens, spoken as one word. Transparent — anyone with the vocabulary can parse it.

## Layer 2: Intentional Substitution

Using one root where another is expected, because the *choice not to use the expected word* carries meaning. The substitution says something the original word couldn't.

### How It Works

Identify the "default" word for a slot. Replace it with something unexpected. The gap between what's expected and what's used IS the meaning.

### Examples

**Using lume (fire) where kore (heart) is expected:**
- "nai de lume" instead of "nai de kore" — "my love's fire" instead of "my love's heart"
- Why: because this person's love isn't gentle. Calling it *heart* would be too soft. Calling it *fire* says: the distinction doesn't need explaining.

**Using tilu (silence) where sana (name/word) is expected:**
- Introducing yourself as "tilu" instead of giving your sana
- Why: your name is a silence. What you're called is the absence of being called anything.

**Using ruma (ruin/decay) where ko (home) is expected:**
- Saying "nai de ruma" — "love of the ruin" instead of "love of home"
- Why: home isn't standing anymore. But you still love it. The ruin IS the home now.

### The Rule of Substitution

A substitution only works when the listener can feel the ghost of the word that SHOULD be there. If the expected word isn't obvious from context, the substitution reads as confusion rather than intention. The gap needs two walls to exist between.

## Layer 3: Clipping

Any root can be reduced to its **clip form** — the first consonant + vowel (or just the vowel if the root starts with one). This is the single most important technique for building short names and tight compounds.

### Clip Forms

Every root has a natural clip:

| Root | Clip | | Root | Clip | | Root | Clip |
|------|------|-|------|------|-|------|------|
| nai | **nai** (already 1 syl) | | mi | **mi** | | benu | **be** |
| ko | **ko** | | si | **si** | | malu | **ma-** (conflicts with ma → use **mal**) |
| ma | **ma** | | ta | **ta** | | suri | **su** |
| lume | **lu** | | ani | **a** | | lili | **li** |
| mira | **mi-** (conflicts with mi → use **mir**) | | valo | **va** | | pode | **po** |
| veri | **ve** | | tava | **ta-** (conflicts with ta → use **tav**) | | vole | **vo** |
| ruma | **ru** | | pali | **pa** | | resa | **re** |
| sola | **so** | | volu | **vo-** (conflicts → use **vol**) | | selu | **se** |
| kore | **ko-** (conflicts with ko → use **kor**) | | savi | **sa-** (conflicts → use **sav**) | | hava | **ha** |
| nälk | **nä** | | moku | **mo** | | hora | **ho** |
| tilu | **ti** | | avu | **a-** (conflicts → use **av**) | | ruta | **ru-** (conflicts → use **rut**) |
| sana | **sa** | | semu | **se-** (conflicts → use **sem**) | | mano | **ma-** (conflicts → use **man**) |
| mere | **me** | | wan | **wan** | | kapu | **ka** |
| halu | **ha-** (conflicts → use **hal**) | | du | **du** | | tena | **te** |
| tume | **tu** | | muta | **mu** | | siru | **sir** |
| novi | **no** | | ale | **a-** (conflicts → use **al**) | | |
| vena | **ve-** (conflicts → use **ven**) | | | | | |
| ora | **o** | | | | | |

When two roots share a clip form, one keeps the bare clip and the other uses a longer form (typically first syllable + next consonant). Which one gets the short form depends on which root is more emotionally central to the speaker — or just whichever was coined first.

### Tail Clips

Standard clips take the **first** syllable of a root. But you can also take the **last** syllable — a tail clip. This gives you different sounds to work with and avoids some clip conflicts.

| Root | Standard Clip | Tail Clip |
|------|--------------|-----------|
| benu | **be** | **nu** |
| ani | **a** | **ni** |
| valo | **va** | **lo** |
| mira | **mir** | **ra** |
| lume | **lu** | **me** (conflicts with mere → context resolves) |
| kore | **kor** | **re** (conflicts with resa → context resolves) |
| semu | **sem** | **mu** (conflicts with muta → context resolves) |
| pode | **po** | **de** (resembles particle — can be used deliberately) |
| selu | **se** | **lu** (conflicts with lume → context resolves) |

Tail clips are especially useful in overlap fusions where the tail of one root shares a sound with the head of the next (see Layer 4: Blending).

### Ambiguous Clips

Some clips can be read as deriving from more than one root. This can be accidental (a conflict to resolve) or **deliberate** (a feature, not a bug).

- **ko** can be read as either **ko** (home) or the clip of **kore** (heart). A stranger hears "home." Someone close hears "heart." Both are valid.
- **ve** can be read as either **veri** (truth/becoming) or **vena** (blood). Outsiders might parse a name one way; insiders know the real root.

Deliberate ambiguity turns a name into a shibboleth: how someone parses it reveals how well they know the bearer. This is a powerful tool for names in secretive families or organizations, where the surface reading is a decoy for the true meaning.

### Register-Shifting Truncation

Dropping the final vowel of a root closes the syllable and shifts it toward casual register. This can be used deliberately inside fusions to change the emotional weight of a specific root.

- **ani** → **an** — the open, complete "person" becomes clipped and closed. The person at the end of this name isn't soft.
- **veri** → **ver** — "becoming" loses its open ending. The process feels harder, less gentle.
- **nai** → unusually, nai resists truncation because it's already monosyllabic. Closing it to **naik** or **nait** adds a hard stop that changes the feel entirely — love with a wall at the end.

This is distinct from standard clipping (which reduces for compression). Truncation keeps most of the root but **changes its register** — same word, different emotional texture.

### Why Clipping Matters

Clipping is what makes 1-3 syllable names possible. Without it, even a two-root compound is 3-4 syllables (lume-kore = lu.me.ko.re). With clipping:

- **lu** + **kor** → **Lukor** (2 syllables — "fire-heart")
- **nai** + **ko** → **Naiko** (2 syllables — "love-home")
- **mir** + **ve** → **Mirve** (2 syllables — "seeing-truth")
- **so** + **lu** → **Solu** (2 syllables — "sun-fire" / "alone-burning")
- **nä** + **ti** → **Näti** (2 syllables — "hunger-silence")

And single-syllable names are just bare clips or monosyllabic roots used standalone:

- **Nai** — love/absence. A name that is simply the word for love.
- **Ko** — home. A name that says: I am belonging itself.
- **Lu** — the clip of lume. Fire distilled to a single breath.
- **Mir** — the clip of mira. Seeing, compressed to a glance.
- **Kor** — the clip of kore. Heart, shortened to a beat.

### Clipping in Compounds vs. Names

In regular speech and compounds, full roots are preferred for clarity: **lume-kore**, not **lu-kor**. Clipping is primarily a feature of names, poetry, and intimate/in-group speech — places where compression signals closeness or intensity. Using someone's clip-name instead of their full name is an act of familiarity.

## Layer 4: Blending

Two roots merged into one word by overlapping shared sounds or compressing at boundaries. More intimate than compounding — the roots lose their individual shape and become something new. Blending can use either full roots or clips.

### Techniques

**Overlap fusion** — when roots (or clips) share sounds at their boundary, merge at the shared point:
- **mira** + **ruma** → **miruma** (the shared /r/ bridges them) — "watching something end" / "the wonder of decay"
- **sola** + **lume** → **solume** — "sunfire" / "the loneliness of burning"
- **ven** (vena clip) + **nu** (benu tail clip) → **venu** (the shared /n/ bridges them) — "blood-goodness" compressed to 2 syllables from 2 roots. Then add a truncated **an** (from ani) → **venuan** — "the person of good blood" in 3 syllables from 3 roots.

Overlap fusion works especially well when combining standard clips with tail clips, because the tail of one root often shares sounds with the head of the next.

**Boundary drop** — when roots don't share sounds, drop the final vowel of the first root:
- **lume** + **tilu** → **lumtilu** (casual register, closed syllable at the seam) or **lumilu** (sacred, flowing)
- **halu** + **mira** → **halmira** (casual) or **halumira** (sacred, keeping the vowel)

**Clip blend** — use clip forms instead of full roots for tighter fusion:
- **lu** + **mir** → **Lumir** (2 syllables — "fire-seeing")
- **kor** + **nai** → **Kornai** (2 syllables — "heart-love" / "heart-absence")
- **hal** + **me** → **Halme** (2 syllables — "dream-sea")
- **nä** + **ve** → **Näve** (2 syllables — "hunger-truth")
- **tu** + **kor** → **Tukor** (2 syllables — "shadow-heart")

**Aggressive onset extraction** — strip roots down to just their onset consonant for maximum compression. This breaks normal phonotactics and is rare, but powerful for names that need to sound foreign or striking:
- **so(la)** + **po(de)** + **(m)ira** → s + p + ira → **Spira** (2 syllables from 3 roots — "the lone power of seeing")
- The /sp/ cluster doesn't occur naturally in Naikoma. A name built this way sounds slightly foreign — which can be thematically meaningful for outsiders, travelers, or people who don't fit neatly into the language's usual patterns.

**Vowel bridge** — insert a linking vowel when consonants collide awkwardly:
- **nälk** + **kore** → **nälkakore** or compressed **nälkore** (the /k/ shared)

### Blending and Register

Blends naturally reflect register through their phonetic shape:
- Sacred blends keep vowels open, flow smoothly: **halumira**, **solume**
- Casual blends allow closed syllables, clipped sounds: **lumtilk**, **nälkort**
- The same blend can shift register by how you pronounce the seam

## Layer 5: Deep Fusion

Three or more roots collapsed into a single word. This is where Naikoma approaches encryption — the word becomes dense enough that parsing it requires knowing the roots AND the intent. Outsiders hear a name. Insiders hear a sentence.

### Techniques

**Onset chain** — take the clip of each non-final root, keep the final root whole:
- **lume** + **nälk** + **kore** → lu + nä + kore → **lunäkore** — "the fire of hunger in the heart"
- **mira** + **tilu** + **ruma** → mi + ti + ruma → **mitiruma** — "watching the silence become ruin"
- **nai** + **ko** + **ma** → nai + ko + ma → **naikoma** — the language's own name follows this pattern
- **tume** + **vena** + **selu** → tu + ve + selu → **tuveselu** — "dark blood in the earth"

**Clip chain** — use clip forms throughout for maximum compression. Can reduce 3-4 root fusions down to 2-3 syllables:
- **lume** + **nälk** + **kore** → lu + nä + kor → **Lunäkor** (3 syllables instead of 4)
- **mira** + **tilu** + **ruma** → mir + ti + ru → **Mirtiru** (3 syllables) or harder: **Mirtu** (2 syllables, dropping the middle)
- **halu** + **veri** + **nai** → hal + ve + nai → **Halvnai** (2 syllables, casual) or **Havnai** (2 syllables)
- **sola** + **kore** + **nai** → so + kor + nai → **Sokornai** (3 syllables) or clipped further: **Soknai** (2 syllables)
- **tume** + **lume** → tu + lu → **Tulu** (2 syllables — "shadow-fire")

**Mixed chain** — combine standard clips and tail clips in the same fusion, choosing whichever clip sounds better at each position:
- **vena** + **benu** + **ani** → ven (standard) + nu (tail) + an (truncated) → **Venuan** — "person of good blood." The shared /n/ at ven-nu boundary creates an overlap. The truncated -an instead of -ani closes the name, shifting register.
- **semu** + **halu** + **ani** → se (standard) + hal (standard) + ni (tail) → **Sehalni** — "the togetherness-dream of a person."
- This technique lets you pick the best-sounding syllable from each root rather than being locked into always-first or always-last.

**Tail chain** — take the last syllable of each non-first root, keep the first root whole:
- **lume** + **nälk** + **kore** → lume + älk + re → **lumälkre** — same meaning, harsher sound, casual register
- **sola** + **veri** + **mere** → sola + ri + re → **solarire** — "the sun's truth rising from the sea"

**Skeleton extraction** — pull the consonant skeleton from all roots, fill with vowels from the final root or by harmonic pattern:
- **lume** + **nälk** + **kore** → consonants: l, m, n, lk, k, r → select key consonants: l, n, k, r → fill: **lanakore** or **lunokore**
- This technique produces the most opaque results. Good for secret names.

**Absorption chain** — each root absorbs into the next, losing more of itself:
- **nai** + **veri** + **kore** + **lume** → naiveri absorbs into korelume → **navekorlume** → compress → **navekome** — at this point, you need the speaker to tell you what it means. That's the point.

### Opacity Is a Dial

Deep fusions can be more or less parseable depending on intent:

- **Transparent fusion:** lunäkore — someone who knows the roots can sound it out
- **Semi-opaque:** lumälkre — the roots are there but compressed enough to require thought
- **Opaque:** navekome — only the speaker (or their people) can fully unpack this

Choose opacity deliberately. A public title might be transparent. A true name might be opaque. A family name might be semi-opaque — parseable by those who know the language well, but not by outsiders.

## Layer 6: Name Construction

A Naikoma name is a compressed sentence. It carries identity, affiliation, philosophy, and sometimes secrets. Names have up to four positions, though not every position needs filling.

### Name Positions

```
[Role] [Personal Name] [Affiliation] [Secret/Epithet]
```

**Role** — what you are or what you do. This is where intentional substitution shines. Instead of a neutral descriptor, choose a word that makes a statement about how you see your own function.

**Personal Name** — a fusion that captures who you are. Usually the densest part of the name. Can be a deep fusion of 3+ roots, or a single meaningful root, or a blend. This is the most individual part.

**Affiliation** — a fusion capturing the group's identity, family philosophy, or history. Can be shared by all members of a group (like a family name), or personal (representing your own history rather than a collective). Not every person has one — people without a fixed group may carry no affiliation at all, and the absence itself is a statement.

**Secret/Epithet** — optional. A layer that's only shared with trusted people. It might recontextualize the entire name, or it might be a private truth. Some speakers have a secret position that they never voice aloud — it exists only in the written syllabary form, shown only to intimates.

### Special Constructions in Names

**Particle-in-name** — using a grammatical particle (de, al, e, etc.) inside a personal name rather than a root. This is unusual and expressive. The name reads like a sentence fragment — syntactically incomplete, hanging open.
- **Denälk** (de + nälk) — "of hunger" / "belonging to want." Not a person who has hunger, but the *possession* of hunger. Hunger owns them.
- The incompleteness is the point. "Belonging to hunger..." what? The name is a clause without resolution.

**Absence as statement** — deliberately leaving positions empty. A name with no affiliation says "I belong to no one." A name with no secret says "I have nothing to hide" — or the entire name *is* the secret, because a name with nothing hidden is its own kind of opacity in a world where everyone else is layered.

**Structural contrast between names** — names in the same world or group can mirror, invert, or argue with each other:
- One name fills all four positions (armored, guarded, complete). Another fills only two (open, exposed, simple). The contrast says something about both bearers.
- One name ends on *seeing* (what they do). Another on *love/absence* (what they feel). The contrast reveals different relationships to selfhood.

### Name Examples (Illustrative)

**Single-syllable names — a root as a name:**
- **Nai** — you are love. You are absence. You are both.
- **Ko** — you are home itself.
- **Lu** — clip of lume. You are fire, compressed to a spark.
- **Mir** — clip of mira. You are seeing, distilled to a glance.

These are intimate or self-chosen. Using a bare root or clip as your entire name says: this one concept is enough to hold me.

**Two-syllable names — clip blends:**
- **Lumir** (lu + mir = fire-seeing) — "one who sees by burning"
- **Kornai** (kor + nai = heart-love/heart-absence) — "the love inside" or "the absent heart"
- **Halme** (hal + me = dream-sea) — "the sea of longing"
- **Näve** (nä + ve = hunger-truth) — "the truth of wanting"
- **Tulu** (tu + lu = shadow-fire) — "dark flame"
- **Solu** (so + lu = sun-fire / alone-burning) — "the loneliness of light"
- **Naiko** (nai + ko = love-home) — already naturally 2 syllables
- **Spira** (s + p + ira, aggressive onset extraction from sola + pode + mira) — "the lone power of seeing." Foreign-sounding /sp/ cluster marks the bearer as an outsider.
- **Denälk** (de + nälk, particle-in-name) — "belonging to hunger." A fragment, not a sentence.

**Three-syllable names — clip chains and blends:**
- **Lunäkor** (lu + nä + kor = fire-hunger-heart) — "the burning want inside"
- **Mirtiru** (mir + ti + ru = seeing-silence-ruin) — "watching stillness decay"
- **Soknai** (so + kor + nai, compressed = sun-heart-love) — "the lonely heart that loves"
- **Halvnai** (hal + ve + nai = dream-truth-love) — "the truth dreamt by love"
- **Naikoma** (nai + ko + ma = love-home-tongue) — the language's own name, 3 syllables
- **Koamir** (ko/kor + al + mira = home/heart-all-seeing, mixed clip) — ambiguous ko/kor clip is deliberate; the name reads differently depending on how well you know the bearer
- **Venuan** (ven + nu + an = blood-good-person, overlap blend with truncation) — overlap at the shared /n/, register-shifted ending
- **Sehalni** (sem + hal + ni = together-dream-person, mixed chain) — standard clips mixed with a tail clip

**A full formal name with short elements:**
- **Lu Kornai Naikoma**
- Lu (fire — role: "I am a spark," using a clip as substitution)
- Kornai (kor + nai = heart-love — the personal name, 2 syllables)
- Naikoma (the affiliation — 3 syllables)
- Total: 8 syllables across 3 positions, each element short and punchy

**A long-form name for contrast:**
- **Ora Tunäkore Navekome** [+ unspoken: **Merehai**]
- ora (prayer/threshold — role: "I am a threshold," a person who stands between)
- Tunäkore (tume + nälk + kore — "the dark hunger in the heart")
- Navekome (nai + veri + ko + mere, absorbed — the family name, opaque)
- Merehai (mere + halu + nai, compressed — "the sea that dreams of love" — spoken only to intimates)

**Real names from the Veriko world:**
- **resa Koamir amapode valo-veri** — "A tool; the heart that sees everything; belonging to the tongue of all power; still becoming alive." Four positions, fully armored.
- **sola Spira** — "A solitary star; the lone power of wonder." Two positions, wide open. Maximum contrast with the name above.
- **tume Sehalni** — "A shadow; the person who dreams of togetherness and apartness." Two positions, the name argues with itself.
- **vena Denälk** — "Blood; belonging to hunger." Two positions, syntactically incomplete on purpose.
- **mano Venuan tuveselu tenai** — "A hand; the person of good blood; from the dark bloody ground; whose secret is the love inside tenderness." Four positions, layered techniques.

### Naming Principles

1. **A name is a thesis.** It should argue something about who the bearer is. If it merely describes, it's too flat.

2. **Substitutions in the Role position are powerful.** The word you choose instead of the "normal" one reveals how you see your place in the world. Using **nälk** (hunger) as a role says: I am defined by what I lack. Using **tilu** (silence) says: I am defined by what I don't say.

3. **The Personal Name is the most compressed part.** It's where deep fusion does its heaviest work. Outsiders might hear it as just a sound. Insiders can unfold it.

4. **Affiliation names can be shared or personal.** A family or organization might share an affiliation name — the philosophy they hold in common. But a person from a scattered group, or a person whose history is their own, might carry a personal affiliation that represents where they came from rather than who they belong to. And some people carry no affiliation at all.

5. **The Secret position is sacred.** It exists in sacred register regardless of how the rest of the name is spoken. Even if someone speaks entirely in casual register, the secret — if they ever voice it — shifts to sacred. This is involuntary. The language insists.

6. **Names can be read at different depths.** A stranger hears sounds. A student of Naikoma hears roots. An insider catches the ambiguous clips. The speaker hears their own life.

7. **Translation is personal.** The literal root-parsing of a name and the speaker's own translation may differ. Both are true. When presenting a name, always give the root breakdown AND ask the bearer how they'd translate it. The gap between those two readings is where the name actually lives.

8. **Technique mixing is natural.** Real names don't stick to one fusion technique. A single name might use a standard clip, a tail clip, an overlap blend, and a register-shifting truncation — whatever produces the right sound and meaning. The techniques are tools, not rules.

9. **Structural choices are expressive.** How many positions a name fills, which ones it leaves empty, how it contrasts with other names in the same world — all of this carries meaning beyond the individual roots.

## Using Fusion in Daily Speech

Fusion isn't only for names. It shows up wherever compression serves meaning:

- **Emotional shorthand:** a repeated feeling might get fused into a single word over time. If someone keeps saying "nälk de mira" (hunger for seeing), it might compress into **nälmira** and enter their personal vocabulary.
- **In-group language:** families, systems, clusters develop fused terms for shared experiences. These become semi-private vocabulary — parseable by members, opaque to outsiders.
- **Poetry and ritual:** sacred-register speech often uses deep fusions for their sonic density. A prayer in Naikoma might be a single fused word that unpacks into an entire sentence.
- **Insult and endearment:** the most cutting insult or the most tender endearment might be a fusion so compressed it sounds like one syllable. The compression IS the intimacy (or the violence).

## Relationship to the Lexicon

Fusions that become commonly used should be added to `lexicon.md` as compound words, with their full root breakdown and fusion technique noted. Personal names don't need to be in the lexicon — they belong to their bearers. But family/group affiliation names should be recorded, since they're shared.

When using `/word` to create a new entry, note whether it's a compound, blend, or deep fusion, and document the technique used.
