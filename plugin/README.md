# Naikoma Language Plugin

A conlang construction and usage toolkit for **Naikoma** — the love-home-tongue, the tongue of the absent home.

Naikoma is a constructed language built from emotional and thematic roots, inspired by toki pona's minimalism and Japanese's layered polysemy. Words carry multiple meanings that shift with context, register, and speaker. The language grows organically through use.

## Commands

- **/word** `[concept]` — Create or define a new Naikoma word. Builds from existing roots first, proposes new ones when needed, and adds approved words to the lexicon.
- **/translate** `[phrase]` — Translate between Naikoma and English (or other languages). Shows root breakdowns, register notes, and polysemous ambiguities.
- **/lexicon** `[search term or "all"]` — Browse or search the living lexicon. Find words by meaning, root, or theme.
- **/speak** `[character] [context]` — Generate in-character Naikoma dialogue. Matches register and voice to the speaker's personality and cluster.

## Core Concepts

- **Polysemy**: Every root carries multiple meanings. Context disambiguates — or doesn't.
- **Register**: Sacred/formal speech uses soft, open sounds. Casual/raw speech adds harder consonants. Same word, different texture.
- **Phonetic echoes**: Roots sound like words from real languages (Japanese, Finnish, Latin, etc.), carrying ghost-meanings.
- **Thematic roots**: Vocabulary is grounded in the emotional architecture of the Initiova system.

## Lexicon

The lexicon file (`lexicon.md`) lives in your workspace folder and grows over time as you create words with `/word`.
